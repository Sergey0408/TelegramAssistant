Заголовок: Создание Telegram-бота для обучения таблице умножения

Описание задачи:
Создать Telegram-бота, который будет помогать пользователям учить таблицу умножения. Бот должен взаимодействовать с пользователями, задавать им вопросы по умножению и проверять их ответы. Также бот должен предоставлять объяснения и советы по улучшению навыков.

подгони размер экрана бота по ширине под размер экрана смартфона.

создай кнопку начать которая соответствует /multiply

Требования:

1. Установка необходимых библиотек:

   • Использовать библиотеку python-telegram-bot для взаимодействия с API Telegram.
при создании библиотек учти проблему которая была в предыдущем проекте .Проблема заключалась в следующем:

    В проекте были ошибки импорта библиотек (LSP Issues):
        "Import telegram could not be resolved"
        "Import numpy could not be resolved"
    Несмотря на то, что библиотеки были указаны в pyproject.toml, они не были корректно установлены.

Чтобы избежать подобных проблем в будущих проектах, следуйте этим шагам:

    Всегда используйте packager_install_tool вместо прямого редактирования pyproject.toml:

packager_install_tool(
    programming_language="python",
    dependency_list=["python-telegram-bot", "numpy", "telegram"]
)

    Убедитесь, что версия Python указана правильно в проекте (в нашем случае Python 3.11)

    После установки зависимостей перезапустите рабочий процесс (workflow), чтобы изменения вступили в силу

    Используйте точные версии пакетов, если это критично для проекта

Этот подход более надежный, чем ручное редактирование pyproject.toml, так как он:

    Автоматически разрешает зависимости
    Создает необходимые файлы конфигурации
    Перезагружает среду после установки



   • Убедиться, что все необходимые библиотеки установлены в среде Replit.

3. Функциональность:

   • Команда /start: Приветственное сообщение с объяснением функций бота.

   • Команда /help: Информация о том, как использовать бота.

   • Команда /multiply: Запуск теста на умножение.

     • В начале игры после приветствия фраза в рамке появляется слово "Продолжить" после нажатия на него .Бот случайным образом выбирает два числа от 1 до 9 и задает вопрос (например, "Сколько будет 3 * 4?"). После 10 правильных ответов. сессия заканчивается и подводятся итоги: Затраченное время, количество ошибок. Эти данные отображаются в рамке. В качестве знака умножить используй букву "x". Середина буквы x по высоте должна совпадать с серединой цифр по высоте.



     • Пользователь вводит свой ответ. Для этого создай клавиатуру с цифрами 0-9 для ввода правильного ответа
если ответ двузначное число бот должен ждать ввода второй цифры


     • Бот проверяет ответ и отвечает:  "Правильно! Молодец!" Далее под этой фразой опять появляется "Продолжить" после нажатия на нее Бот случайным образом выбирает два числа от 1 до 10 и задает вопрос (например, "Сколько будет 3 * 4?") и так далее до 10 правильных ответов.

       • Если ответ неправильно,отвечает Ошибка и появляется надпись "Запомомни правильный ответ" и сам правильный ответ в виде примера с ответом. Далее появляется "Продолжить" после нажатия на нее Бот случайным образом выбирает два числа от 1 до 10 и задает вопрос (например, "Сколько будет 3 * 4?") Если ответ правильный, то поздравляет пользователя и дает последний пример ответ на который был неправильный. Перед примером оповещает "Давай повторим сложный пример". Если повторно ответ неправильный, то этот пример педлагается еще раз. и так до тех пор пока не будет получен правильный ответ.
